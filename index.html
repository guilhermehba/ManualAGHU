Caminho até o cadastro de Internações

Este módulo envolve a navegação no menu de internação que levam até a interface de cadastro de internação de pacientes.
No Menu lateral Esquerdo há uma série de campos selecionáveis com icones de identificação. Clique no campo chamado "INTERNAÇÃO"
Ao fazer isso, será aberto um pequeno menu logo abaixo de "INTERNAÇÃO" com algumas opções. Agora Clique em "CADASTRO"
Dentro de cadastro será exibida uma nova lista de campos selecionáveis. Clique em "UNIDADES FUNCIONAIS"
Agora clique em "PESQUISAR", um botão com um icone de lupa à esquerda. Assim serão exibidas todas as unidades funcionais cadastradas até o momento.
Caso a unidade desejada não esteja entre elas, clique no botão "NOVO" que apareceu ao lado do botão limpar.
AGORA PROXIGA PARA A PROXIMA ETAPA DO TUTORIAL.

Campos com um Asterisco "*" ao lado do titulo, são campos de preenchimento obrigatório!

Cada campo possui um ponto de interrogação "?" ao lado do titulo, que mostra uma rápida descrição daquele campo!

Módulo de Internação

Este módulo envolve o ato de registrar a entrada do paciente na estrutura hospitalar, relacionando-o a um leito, especialidade, médico responsável, CID de internação, procedimento a ser realizado e outros dados necessários para efetivar o processo de internação administrativa.
Após preencher todos os campos de dados de cadastro, será a vez de selecionar qual unidade que irá efetuar o atendimento.
selecione APENAS "Unidade de Ambulatorio"
Concluído esta etapa, proxiga para o próximo tutorial. Clique em "AVANÇAR"

Caminho até Grade de Agendamento Ambulatorial

Este módulo envolve a navegação no menu de internação que levam até a interface de cadastro Ambulatorial.
No Menu lateral Esquerdo há uma série de campos selecionáveis com icones de identificação. Clique no campo chamado "AMBULATÓRIO"
Ao fazer isso, será aberto um pequeno menu logo abaixo de "AMBULATÓRIO" com algumas opções. Agora Clique em "GRADE DE AGENDAMENTO"
